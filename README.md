# Neighbourhood
#### Django app that users to be in the loop about everything happening in the neighborhood., 26-09-2018
#### By **Kabage Mark Chege**
## Description

This is a web app that allows one to be in the loop about everything is happening in the neighbour hood. Nobody is ever aware on anything that is happening in the neighbourhood therefore this web app
intends to solve that.registered users will be required to fill their locations, different businesses
,Find Contact Information for the health department and Police authorities near my neighborhood and create Posts that will be visible to everyone in my neighborhood.  


## Setup
* clone the repostory with **git clone 'URL'**
* Activate virtual with **source virtual/bin/activate**
* Start the server with **python3.6 manage.py runserver**
* click the url to see app on browser **http://127.0.0.1:8000/**

## Installation Requirements
* Web Browser
* python Django Framework
* Virtual enviroment
* Text Editor

## Known Bugs
